# Sample Pairs

Use this folder for demo-ready audio pairs so you can run the app without uploading files.

## Folder Structure (used directly by the app)

Create one subfolder per pair, with:

- `a.wav`
- `b.wav`

Example:

```text
demo/samples/
  pt1_same/
    a.wav
    b.wav
  pt1_different/
    a.wav
    b.wav
```

If folder names end with `_same` or `_different`, the app shows that label in the dropdown.

Supported formats: `.wav`, `.flac`, `.mp3`, `.m4a`
